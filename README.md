# pi-pipeline

Extension for Pi coding agent.

## Install

### From npm (recommended)

```bash
pi install npm:pi-pipeline
```

### From GitHub Packages

```bash
pi install npm:@baphuongna/pi-pipeline
```

### From local source (development)

```bash
cd pi-pipeline && pi install .
```

## Verify

```bash
pi list
```
